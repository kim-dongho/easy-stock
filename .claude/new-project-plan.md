# 신규 프로젝트 기획 — 주식 매수 추천 사이트

## 컨셉

- 주식 초보도 이해할 수 있는 매수 추천 + 분석 사이트
- 기술 용어를 쉬운 말로 번역해서 보여주는 UX
- 한국 + 미국 주식 둘 다 지원

## 기술 스택

- **Next.js** — 프론트엔드
- **Go** — API 서버 (DB 읽기 전용)
- **PostgreSQL (TimescaleDB)** — 기존 quant-platform DB 공유 (SELECT only)
- Python 엔진 불필요 — 데이터는 기존 프로젝트에서 적재

## DB 공유 구조

```
기존 quant-platform (데이터 생산)
  engine(Python) → market_data, factors, candles_1h 적재
                         ↓ (읽기 전용)
새 프로젝트 (데이터 소비)
  Next.js → Go API → PostgreSQL (같은 DB)
```

### 사용 가능한 factors 테이블 컬럼

- `sma_50`, `sma_200` — 이동평균선
- `price_vs_sma50`, `price_vs_sma200` — 이동평균 대비 가격
- `sma20_vs_sma50` — 이동평균 교차
- `rsi_14` — RSI
- `vol_ratio_20d` — 거래량 비율
- `return_5d` — 5일 수익률

## 핵심 기능 (1차)

1. **오늘의 추천 리스트** — 매수 추천 종목 카드형 리스트, 추천 이유를 쉬운 말로
2. **종목 상세 페이지** — 차트 + "지금 왜 주목받나?" 설명
3. **투자 용어사전** — 사이트에서 쓰는 용어를 초보 눈높이로 설명
4. **시그널 가이드** — "이런 패턴이 나타나면 이런 의미예요"

## 매수 추천 로직

### 개인 시그널 (SMA 50/200 기반)

- **골든크로스 임박** — SMA50이 SMA200 아래에서 위로 올라오기 직전 (갭 -2%~0%)
- **200일선 지지** — 가격이 SMA200 근처 (±1%)에서 반등

### 복합 점수 시스템

| 시그널           | 조건                         | 가중치 |
| ---------------- | ---------------------------- | ------ |
| 골든크로스 임박  | SMA50/SMA200 갭 -2%~0%       | 높음   |
| 200일선 지지     | price_vs_sma200 ±1%          | 높음   |
| RSI 과매도 반등  | rsi_14 30~40 (바닥 탈출)     | 중간   |
| 거래량 증가      | vol_ratio_20d > 1.5          | 중간   |
| 단기 모멘텀 전환 | return_5d > 0 (하락 후 반등) | 낮음   |

여러 시그널이 겹칠수록 추천 점수가 높아지는 구조.

### 초보 친화 표현 예시

```
❌ "SMA50/200 골든크로스 임박, RSI 35 과매도 반등"

✅ "📈 장기 상승 추세로 전환 중이에요
    주가가 장기 평균선(200일)을 뚫고 올라가려는 움직임이에요.
    최근 많이 빠져서 반등할 가능성이 높아요."
```

## 2차 기능 (나중에)

- **애널리스트 목표가 / 컨센서스** 연동
  - 미국: FMP API ($14/월) — 목표가, 실적 추정치, 재무제표, 뉴스
  - 한국: KRX 정보데이터시스템 (무료) + 한경 컨센서스 (크롤링)
- 실적 서프라이즈 알림
- 재무제표 요약 (PER, PBR, ROE)
